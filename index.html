Mes HSC forward WMC fair value Swap cotizado Volumen Costo fair value Costo swap
Fair value WMC esperado = hedge ratio × HSC forward + intercepto, según la regresión de 36 observaciones (R²=0.9850). El margen objetivo se deriva del Standard Error de esa misma regresión (k × SE): k=1 tolera hasta una desviación estándar de ruido normal del modelo (~68% de los casos históricos), k=1.65 ~90%, k=2 ~95% — entre más alto el k, menos falsas alarmas pero más lento para detectar un margen real que te esté cobrando el dealer. El margen implícito es la diferencia entre el precio de swap cotizado y ese fair value ponderado por volumen; si queda por debajo de tu margen objetivo, la cotización está a tu favor. Actualiza la curva forward con los valores reales de CME/EIA para cada strip que evalúes.
